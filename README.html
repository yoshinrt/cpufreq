1. CPUFreq とは?

　CPUFreq とは，CPU のクロック数を計測しタスクバーアイコンとして表示する，Windows 用常駐プログラムです．ウィンドウなどを作らず，邪魔にならないのが特徴です．数字の色も 16色の中から選択可能です．
　CPU のクロックを計るといっても，ベンチマークのような CPU をフル稼働させるようなものではなく，10秒毎に TSC というレジスタを読み出すだけなので，CPU にかかる負荷は，時計プログラム並に軽いです．
　Pentium 以降の CPU 必須です．

2. CPUFreq の使用法

アーカイブの内容
CPUFreq.exe 実行プログラム本体
ReadMe.html マニュアル ( このファイル )

　ただ CPUFreq.exe を実行するだけです．(^_^;) すると，画面の右下 ( タスクバーが下にある場合 ) に CPU の周波数がデジタル数字で表示されます．デジタル数字をクリックすると，メニューが現れます．

color Setting....
「color Setting....」を選ぶと，デジタル数字の色を選ぶダイアログが表れます．ただし，Windows 標準の 16色 ( 要するに基本的な色 ) しか使用することはできません．

Close
「Close」を選ぶと，CPUFreq を終了します．

　また，デジタル数字の上にカーソルを持ってくると，0.001MHz 単位の正確なクロック数を表示します．

3. アンインストール

  1. アーカイブを解凍して出来たファイルをすべて削除してください．
  2. レジストリエディタで，「\HKEY_CURRENT_USER\Software\DeyuDeyu\CPUFreq」以下を削除してください．色の設定を変更していない場合は，このキーがレジストリに作られていない場合もあります．

4. バグ等

  • デジタル表示は，タスクバーアイコンを 2個並べて表示させているので，タスクバーが画面上か下以外 ( すなわち，タスクバーを立てて使っている ) の場合は，デジタル表示がずれます．
  • 486 以前の CPU ではサポートしていない命令 ( rdtsc ) を使っていますので，486 以前の CPU ではお使いになれません．( CPU 種識別をサボっているため，不正終了します (^_^;) )
  • デスクトップ PC など CPU 周波数が変化しないはずの PC でもクロック数表示の下の桁が細かく変動しますが，これは Windows のタイマー精度が悪いせいです．
